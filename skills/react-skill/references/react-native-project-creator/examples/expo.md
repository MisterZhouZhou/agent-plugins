# Expo Project Creation

## Use for

New React Native projects that benefit from Expo tooling, managed services and simpler setup.

## Command

```bash
npx create-expo-app MyApp --template blank-typescript
```

## After creation

```bash
cd MyApp
npm start
```

## Checks

- Does the app need custom native modules unsupported by managed Expo?
- Which package manager does the workspace use?
- Is the first generated app running before dependencies are added?
