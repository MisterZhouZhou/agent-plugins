# Core APIs

- `View` - layout container.
- `Text` - text rendering.
- `Pressable` - touch interactions.
- `TextInput` - input fields.
- `FlatList` - virtualized lists.
- `ScrollView` - small scrollable content.
- `Image` - images.
- `StyleSheet` - style definitions.
- `Platform` - platform detection and selection.
- `Dimensions` and `useWindowDimensions` - viewport sizing.
- `SafeAreaView` - safe area handling when appropriate.

## Checks

- Avoid web-only DOM APIs.
- Keep touch targets large enough for mobile.
- Test native behaviors on physical devices or simulators when possible.
